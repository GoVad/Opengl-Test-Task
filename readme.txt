Приложение написано для android API 30, я использовал девайс с разрешением 480х800.
В качестве OpenGL фреймворка использовался OpenGL ES 2.0, соответственно везде используется класс GLES20.
Класс MainActivity содержит в себе создание элементов GUI и добавление к ним слушателей.
Внутри себя он реализует обьект класса MyGLSurfaceView, который используется для настройки версии Opengl и создания обьекта
класса для рендера MyGLRenderer. Так же MyGLSurfaceView используется для связи GUI и класса для рендера.
MyGLRenderer реализует в себе всю основную логику связанную с отрисовкой сцен, созданием фреймбуферов и тд.
MyGLUtils класс с только статическими функциями, которые используется для различных утилитарных действий.
Класс PostRenderer используется для постобработки изображения и наложения на него различных фильтров, например яркость,
блюр и виньетка.
Класс Primitive используется для рисования примитивных фигур с набором из углов заданного цвета.
ProgrammClass содержит в себе шейдерные программы (де-факто лишь одну для примитивов, но туда можно вынести программу из
PostRenderer)
